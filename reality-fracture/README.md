# Stranger Things Trailer Experience

## How to Add Your Images

Place the following images in the `assets` folder with these exact names:

1. **`creel-mansion.jpg`** - The Creel Mansion under red sky with blue lightning
2. **`missing-boy.jpg`** - The newspaper collage about the missing boy (Will)  
3. **`radio.jpg`** - Radio with red heartbeat line
4. **`vecna-portal.jpg`** - Vecna emerging from the red portal (the wide cinematic scene)
5. **`eleven-confrontation.jpg`** - Eleven vs Demogorgon confrontation

## How to Use

1. Open `index.html` in your browser
2. The trailer will auto-play through all scenes
3. Use **← →** arrow keys or click navigation buttons
4. Press **SPACE** for next scene
5. Press **ESC** to skip to finale
6. Swipe left/right on mobile
